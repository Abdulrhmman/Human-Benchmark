# Human-Benchmark
<p>Human Benchmark android application is a simple game where you can measure your intelligence with fun games.</p>

<strong>The game has six types of tests:</strong>

<ul>
  <li>Reaction Time</li>
  <li>Aim Trainer</li>
  <li>Number Memory</li>
  <li>Verbal Memory</li>
  <li>Chimp Test</li>
  <li>Visual Memory</li>
</ul>
